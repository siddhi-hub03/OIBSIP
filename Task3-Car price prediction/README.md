# 🚗 Car Price Prediction Using Machine Learning

## 📌 Project Overview

Car price depends on various factors such as brand value, manufacturing year, mileage, fuel type, transmission, ownership, and current market price.

This project aims to build a Machine Learning model that predicts the selling price of a used car based on different features using regression algorithms.

The project uses a **Random Forest Regression model** to estimate the car selling price accurately.

---

## 🎯 Objective

- Analyze factors affecting car prices
- Perform data preprocessing and feature engineering
- Train a machine learning regression model
- Predict the selling price of a car based on user inputs

## 🤖 Machine Learning Model

### Random Forest Regression

Random Forest was used because it:
- Handles non-linear relationships well
- Works effectively with mixed features
- Provides good prediction accuracy

---

## 📊 Model Evaluation

Evaluation metrics used:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
